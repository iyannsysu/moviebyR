=========================================
Movie Free by R
=========================================
Website Nonton Movie & Series Streaming Gratis
Powered by MovieBox API (Sonzai X API)

FITUR LENGKAP:
- Homepage dengan Banner Slider otomatis
- 6 Kategori Movie: Global, Indonesia, Hollywood, Asia, Horror, Animasi
- 6 Kategori Series: K-Drama, Indo Drama, Anime, Barat, C-Drama, Thai-Drama
- Detail Movie lengkap: sinopsis, cast, genre, IMDb rating, dll
- Pencarian film/series
- Video Player streaming
- Download Movie multi-resolusi (360P, 480P, 720P, 1080P)
- Episode list untuk Series + streaming per episode
- Pagination, Responsive Design
- Auto-update data dari API

STRUKTUR FILE:
index.html      - Halaman utama
css/style.css   - Semua styling
js/config.js    - Konfigurasi API
js/app.js       - Aplikasi JavaScript
